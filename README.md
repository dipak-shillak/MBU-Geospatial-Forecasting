# AI Driven Geospatial Forecasting of MBU Demand

UIDAI Hackathon 2026 Project

## Problem Statement
Uneven demand for Mandatory Biometric Updates causes congestion in some regions while others remain underserved.

## Solution
An AI driven geospatial forecasting system that predicts district wise MBU demand and visualizes it on an interactive map.

## Features
- District wise demand prediction
- Geospatial heatmap visualization
- Inclusion Difficulty Index
- Resource planning support

## Tech Stack
- Python
- Pandas NumPy
- Scikit Learn
- Folium
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
